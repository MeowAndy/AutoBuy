rocket_launch 快速开始

1

安装依赖

确保已安装 Python 3.8+，并安装项目依赖

uv sync
2

启动应用

运行以下命令启动 Web 服务

uv run app.py
3

访问界面

在浏览器中打开 Web 界面

http://localhost:5000 （或你启动时指定的端口，例如 5001 / 5002）

store 平台使用说明

京东抢购

  • schedule 精准定时：使用京东服务器时间进行抢购
  • auto_fix_high 自动操作：到达时间自动点击提交订单
  • verified 登录确认：扫码后点击确认按钮继续
TB

淘宝抢购

  • schedule 精准定时：使用淘宝服务器时间进行抢购
  • touch_app 手动准备：需手动勾选商品并进入结算页
  • speed 高频点击：多线程快速点击提交订单
warning

注意事项

  • 请确保已安装 Chrome 浏览器
  • 首次运行可能需要下载 ChromeDriver（程序会自动处理）
  • 抢购前建议提前测试登录和页面跳转流程
  • 抢购成功后请尽快完成付款
  • 请勿用于违规抢购，仅限学习交流使用

quiz 常见问题

浏览器启动失败怎么办？

请检查是否安装了 Chrome 浏览器，或尝试更新 ChromeDriver 版本。程序会自动尝试下载匹配的 ChromeDriver。

登录二维码扫描后没反应？

请确保网络连接正常，扫码成功后需要在页面上点击"确认登录"按钮才能继续流程。

抢购时间已到但没有自动提交？

请检查时间格式是否正确，确保已完成"确认购物车"步骤。可查看实时日志了解具体执行状态。
