# 更新日志

## F0.0.1

本版本聚焦于 **多实例运行体验** 与 **可维护性**。

### 新增
- 支持通过 `python app.py --port <端口>` 启动不同实例
- `start.bat` 支持：
  - 直接输入端口启动
  - 回车后从 `5000` 开始自动寻找空闲端口
  - 启动后自动打开对应浏览器地址
- 新增页面内 **更新脚本** 按钮
  - 点击后自动执行 `git pull`、安装依赖并重启当前实例

### 优化
- 改善多开场景下的实时日志同步
- 当前端 SSE 流遗漏日志时，前端会通过状态轮询自动补齐日志
- 任务进入成功状态时，页面会自动同步到“抢购完成”阶段并启用相关按钮

### 兼容性说明
- 保持原有 `start.bat 5001` 这类命令行传参方式不变
- 不影响单实例默认使用 `5000` 端口的习惯
