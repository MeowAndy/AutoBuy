# 🛒 AutoBuy（小白友好版）

给 **Windows 新手** 的淘宝/京东抢购辅助工具。  
你只要按页面步骤点按钮，程序会自动帮你走到“提交订单”前后；**付款仍需你自己操作**。

## 你适合用这个吗？

适合：
- 想抢购，但不想写代码
- 想看得见日志、看得懂步骤
- 在 Windows 电脑上使用

不适合：
- 想全自动付款（本工具不做支付）

## 1 分钟上手

### 第一步：准备环境
- 安装 Python 3.8+
- 安装 Chrome 浏览器

### 第二步：启动
双击项目里的：`start.bat`

> 如果弹出端口输入，直接回车就行。

### 第三步：打开页面
浏览器访问：`http://localhost:5000`

## 抢购怎么用（照着做）

1. 选择平台（淘宝/京东）
2. 设置抢购时间
3. 点“开始抢购”
4. 在弹出的浏览器里扫码登录
5. 回页面点“确认登录”
6. 进入购物车选好商品后点“确认购物车”
7. 到点自动提交订单
8. 手动付款

## 常见问题（小白必看）

- **Q：抢购失败怎么办？**  
  A：先校准系统时间，提前 3~5 分钟启动，再试一次。

- **Q：可以多开吗？**  
  A：可以。用不同端口启动：
  ```bash
  python app.py --port 5001
  python app.py --port 5002
  ```

- **Q：我不懂驱动怎么办？**  
  A：不用手动装，程序会自动处理 ChromeDriver。
