# 🛒 AutoBuy（Windows 场景）

面向 **Windows 本地桌面** 的淘宝 / 京东抢购辅助工具。  
核心思路：你在网页里点流程，程序用 Selenium 在浏览器执行；**付款仍需手动完成**。

## 这个项目适合谁

- 想在 Windows 上做抢购自动化（而不是写接口脚本）
- 需要可视化页面 + 实时日志
- 需要多开实例（不同端口）

## 快速开始（3 分钟）

1. 安装 Python 3.8+
2. 安装 Chrome
3. 在项目目录执行：

```bash
pip install -r requirements.txt
python app.py
```

或直接双击：

```bat
start.bat
```

打开浏览器访问：`http://localhost:5000`

## 典型使用流程

1. 选平台（淘宝/京东）
2. 设定抢购时间
3. 点击开始，扫码登录
4. 手动进购物车并确认目标商品
5. 到点自动提交订单
6. 手动支付

## 多开实例（同机多任务）

```bash
python app.py --port 5001
python app.py --port 5002
```

`start.bat` 也支持输入端口；端口占用时会自动找可用端口。

## 注意事项

- 仅适用于可进入提交订单页的商品
- 建议提前 3~5 分钟启动
- 抢购期间不要关闭程序窗口
- 该工具用于学习与效率辅助，请遵守平台规则
